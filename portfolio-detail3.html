Mashup_Game for fun

The Mashup Game Website is a full-stack web application that blends creativity and technology, built with React (TypeScript) on the frontend and Node.js with Express on the backend. It allows users to log in securely using JWT authentication, choose game genres, and generate unique character mashups through an animated spinning wheel. Players can view their results on interactive 3D cards, upload their mashup artworks, and showcase them in a community gallery where others can like and comment. The system also includes a personalized dashboard for users to manage their profiles and view their statistics.

Technically, the project demonstrates advanced React hooks usage such as useState, useEffect, useContext, and useRef for state, side-effects, global context, and performance optimization. It connects to a MySQL database through XAMPP and handles secure data flow with bcryptjs for password hashing and Multer for file uploads. With its structured backend, real-world authentication flow, and responsive Tailwind CSS design, this project serves as a practical learning example for mastering full-stack React development and understanding how frontend and backend interact in a production-style environment.

I faced some challenges in aligning the design with my vision, but through persistence and consistency, I was able to create a dynamic genre character system. This system allows characters to change through the admin panel, similar to how a restaurant menu can be updated, making the application interactive rather than static. The project taught me a great deal about Node.js, managing CRUD operations, and handling data through an admin interface to make the application more dynamic and user-friendly.
